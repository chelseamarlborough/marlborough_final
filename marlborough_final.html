Introduction

The first piece of this paper was a replication of Stokes’ results. Here code and data are publically available on the Harvard Dataverse. In order to replicate her results, I ran her orignial code using R code. I was easily able to run the code for both the tables and figures, but the code for formatting the tables in LaTex was not provided on Dataverse. I also was unable to create Figure 1, as code was also not provided. Instead, I opted to analyze the raw output of the regressions from my replications. All code for the replication is available in my GitHub repository.

After the replication, in my extension of Stoke’s work I test the strength of Stokes’ model by running a Bayesian regression upon it. In order to perform this treatment effect, I created a new variable named “effect” that is deprived from “perc_lib”. According to Stokes’ codebook, “perc_lib” represents the votes cast for the Liberal Party divided by the number of voters who cast ballots in the precinct. In other words, “perc_lib” is equivalent to vote share. If the vote share is greater than 0, it is equal to 1 in “effect”. If the vote share is less than 0, it is equal to 0 in “effect”. I then replaced Stokes’ linear regressions with Bayesian ones and replaced “perc_lib” with “effect”.
